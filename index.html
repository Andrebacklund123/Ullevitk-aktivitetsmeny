“Fem nivåer, oändliga möjligheter – för spelare på alla nivåer mitt i centrum.”

Utforska aktiviteter

Grön Nivå (1/5)

Grön nivå passar dig som är nybörjare eller har spelat väldigt lite tidigare. Fokus ligger på att lära sig tennisens grunder, som grepp, svingar, fotarbete och bollkontroll. Målet är att du ska kunna spela med en annan spelare.

  • Klicka för att boka Tennispasset Grön
  • Klicka för att boka Nybörjarkurs

Gul Nivå (2/5)

På gul nivå förväntas du ha en grundläggande förståelse för tennis. Du kan hålla igång spelet själv, delta i enklare matcher och har grundläggande kunskap om taktik och rörelsemönster på banan.

  • Klicka för att boka Tennispasset Gul
  • Klicka för att boka Fortsättningskurs
  • Klicka för att boka Fotarbete Gul/Blå
  • Klicka för att boka Teknik Grundslag, Gul/Blå
  • Klicka för att boka Gul 90 minuter

Blå Nivå (3/5)

Blå nivå är för dig som behärskar spelet och kan röra dig effektivt över hela banan. Du kan kontrollera tempot, placera bollen med precision och använda olika typer av spinn i dina slag.

  • Klicka för att boka Tennispasset Blå
  • Klicka för att boka Blå 90 minuter
  • Klicka för att boka Fotarbete Gul/Blå
  • Klicka för att boka Teknik Grundslag, Gul/Blå

Röd Nivå (4/5)

På röd nivå spelar du på ett högt tempo med få misstag och kan hantera matcher mot olika typer av motstånd. Du varierar både tempo och spinn och har den fysiska förmågan att täcka hela banan. För att spela på röd nivå behöver du rekommenderas av en tränare.

  • Klicka för att boka Tennispasset Röd
  • Klicka för att boka Fotarbete Röd/Svart
  • Klicka för att boka Röd 90 minuter

Svart Nivå (5/5)

Svart nivå är för de mest avancerade spelarna, ofta tidigare tävlingsspelare eller personer med lång erfarenhet av tennis. Träningen fokuserar på detaljer snarare än grunder och följer en liknande struktur som för klubbens tävlingsjuniorer. Även här krävs en rekommendation från en av klubbens tränare för att delta.

  • Klicka för att boka Tennispasset Svart
  • Klicka för att boka Svart 90 minuter
